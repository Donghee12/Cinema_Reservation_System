# 🎬 콘솔 기반 C++ 영화 예매 시스템

영화 좌석을 선택하고 예약, 결제, 취소까지 가능한 콘솔 애플리케이션입니다. 예약자는 날짜, 시간, 인원수, 영화 제목, 좌석을 선택하고 예매를 완료할 수 있으며, 차량 번호 입력 및 주차 시간 관리 기능도 포함됩니다.

## 🧰 기능 소개

- 🎫 **영화 예매**  
  날짜, 시간, 인원수 선택 → 영화 선택 → 좌석 예약 → 결제 → 예매 정보 저장

- 🧾 **예매 내역 확인 / 취소**
  - 사용자가 입력한 이름 기반으로 `.txt` 파일을 검색해 예매 내역 확인
  - 취소 시 좌석 정보도 자동 갱신

- 🅿 **차량 번호 등록 및 확인**
  - 예매자만 3시간 무료 주차 가능
  - 닉네임과 차량 번호 연동

- 📽 **영화 포스터 슬라이드**
  - 이미지 간 fade 전환 구현

## 📁 파일 구성

```
.
├── main.cpp                     # 메인 메뉴 및 흐름 제어
├── reservation.cpp             # 영화 목록 및 예매자 정보
├── seat.cpp                    # 좌석 선택, 결제, 파일 저장
├── cancelBooking.cpp           # 예매 취소 기능
├── viewBookingInfo.cpp         # 예매 내역 보기
├── CarNumberInput.cpp          # 차량 번호 입력
├── CarNumberOutput.cpp         # 차량 번호 출력
├── D_T_P.cpp                   # 날짜/시간/인원 통합 처리
├── DateSelector.cpp            # 날짜 선택
├── TimeSelector.cpp            # 시간 선택
├── PeopleSelector.cpp          # 인원 선택
├── slide_img.cpp               # 영화 포스터 전환
├── images/                     # 영화 포스터 이미지 (1.jpg ~ 5.jpg)
├── *.txt                       # 예매 결과 저장 파일들
```

## 🛠 주요 기능 설명

| 기능명              | 설명 |
|-------------------|------|
| `startBookingProcess` | 좌석 예매 + 결제 처리 루프 |
| `cancelBooking`       | 예약된 텍스트 파일 검색 → 삭제 및 좌석 반영 |
| `viewBookingInfo`     | 이름 기반으로 예매 파일 목록을 보여주고 출력 |
| `CarNumberInput` / `Output` | 차량 번호 입력 / 출력 (OpenCV UI 포함) |
| `slide_img`           | 영화 포스터 전환 애니메이션 |

## 🔄 예시 흐름

1. 프로그램 실행
2. [1] 예매 선택 → 아이디 입력
3. 영화 선택
4. 날짜, 시간, 인원 선택
5. 좌석 선택 및 결제
6. 예매 정보 파일 자동 저장
7. [2] 예매 내역 확인 또는 [3] 예매 취소 가능
8. 차량 번호 입력 (자동 주차권 발급)
